# Iot-precision-agriculture-system
IoT-based automated system for soil digging, seed sowing and fertilizer dispensing using sensors and microcontroller.
Automation: Designed a rover capable of automating soil digging, seed sowing, and fertilization to minimize manual labor and human error.
​Customization: Implemented three fixed modes for seed depth and spacing, along with a customizable mode for user-defined parameters via a mobile interface.
​Technical Workflow: Programmed the rover to execute linear movement, precise stopping, and simultaneous dispensing of seeds and fertilizers.
​Results: Achieved 98% precision in operational tasks, providing an affordable and scalable automation solution for sustainable farming.
